# :ocean: Profile 포트폴리오

간단한 소개와 보유한 능력과 작업물들의 정보를 담고 있는 프로필 포트폴리오입니다. 
<br/><br/><br/><br/>


## :smirk: '저는 이런 사람입니다'

해당 작업물은 정성과 시간을 들여 만든 결과물들을 담고 있는 웹페이지입니다. 

반응형 웹으로 제작됐으며 HTML과 SASS로 모양을 잡고 자바스크립트와 제이쿼리를 사용하여

기능을 추가했습니다.

<br/><br/>
